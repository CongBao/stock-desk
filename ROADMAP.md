# Roadmap

This roadmap communicates direction, not a delivery guarantee. Scope and sequencing may change after technical validation and user feedback.

| Stage                    | Status  | Intended outcome                                                                                                                                                                                          |
| ------------------------ | ------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 0 — Foundation           | Current | Local FastAPI/SQLite service, durable task worker, React shell, encrypted secret storage, reproducible native/container gates, and public project governance. Product routes remain previews where noted. |
| 1 — Market data          | Planned | Provider adapters, normalized A-share instruments and bars, cache/update policies, and real chart rendering with provenance.                                                                              |
| 2 — Formulas             | Planned | A constrained formula language, validation, execution, indicators, and signal visualization with compatibility documented by tests.                                                                       |
| 3 — Backtests            | Planned | Reproducible daily, weekly, and 60-minute simulations with explicit fees, corporate-action treatment, metrics, and exportable results.                                                                    |
| 4 — Intelligent analysis | Planned | User-configured domestic model providers, auditable research workflows, citations, cost controls, and human confirmation for consequential actions.                                                       |
| 5 — Integrated desk      | Planned | Coordinated analysis agents, monitoring, portfolio context, and a cohesive decision workspace with stronger operational hardening.                                                                        |

No later-stage capability is considered complete merely because its navigation route or layout preview exists.
