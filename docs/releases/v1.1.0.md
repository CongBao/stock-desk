# Stock Desk v1.1.0

[English](#english)

## 中文

`v1.1.0` 是面向 Windows x64 的桌面体验正式版本，重点改善安装后即可使用的完整体验：

- 独立桌面窗口、普通用户安装和随应用启停的受控本地服务；
- 首次使用向导、可搜索股票入口，以及未选择时默认打开上证指数 `000001.SS`；
- 宽屏、窄屏、短窗口和 200% 等效缩放下不重叠的响应式布局与图标侧栏；
- 通达信兼容公式、日线/周线/60 分钟的 MACD 或自定义买卖信号回测；
- 退出确认、安全检查点、中断任务恢复，以及可选的仅 v1.1 本地数据卸载；
- 复用受保护 `main` 已通过 CI 的 exact-SHA Windows 候选，不在发布阶段重建或重复运行测试。

本版本是 **unsigned / 未签名** 版本，没有 Authenticode 证书，也不是自签名版本。Windows 可能显示“未知发布者”或 SmartScreen 提示；安装前请核对发布页中的 `UNSIGNED-WINDOWS-SHA256SUMS`、候选清单、builder provenance 和从同一候选目录生成的 SPDX SBOM。SignPath Foundation 因项目曝光率不足拒绝了免费签名申请。生产 updater 在 v1.1.0 中继续关闭，软件不会在后台下载或安装更新。

计划中的 v1.2 将通过 Microsoft Store / MSIX 获得商店签名与分发信任。v1.1.0 不包含 macOS、Linux、Android 或 ARM64 安装包，不连接券商、不执行交易，也不会自动迁移或删除旧版 v1 数据。

## English

`v1.1.0` is the Windows x64 desktop-experience release. It focuses on a usable install-and-open journey:

- a standalone desktop window, per-user installation, and a controlled local service that follows the app lifecycle;
- first-run setup, searchable stock selection, and Shanghai Composite `000001.SS` as the no-choice default;
- responsive, non-overlapping layouts for wide, narrow, short, and 200%-equivalent views with icon navigation;
- TDX-compatible formulas and daily/weekly/60-minute backtests driven by MACD or custom buy/sell signals;
- exit confirmation, safe checkpoints, interrupted-work recovery, and optional removal of v1.1-only local data;
- exact reuse of the Windows candidate proved by protected-`main` CI, with no release-stage rebuild or repeated source tests.

This is an **unsigned** release. It has no Authenticode certificate and is not self-signed. Windows may show an Unknown Publisher or SmartScreen warning. Before installation, verify `UNSIGNED-WINDOWS-SHA256SUMS`, the candidate manifest, builder provenance, and the SPDX SBOM generated from that same candidate directory. SignPath Foundation rejected the free-signing application because the project did not yet have enough exposure. The production updater remains disabled in v1.1.0 and will not download or install updates in the background.

The planned v1.2 will target Microsoft Store / MSIX distribution for Store signing and trust. v1.1.0 contains no macOS, Linux, Android, or ARM64 package, does not connect to a broker or place trades, and does not automatically migrate or delete legacy v1 data.
