# Stock Desk v1.1.0-beta.2

[English](#english)

## 中文

`v1.1.0-beta.2` 是仅面向 Windows x64 的桌面体验预发布版本。它在现有 Tauri 独立窗口、受控 Python sidecar 和普通用户 NSIS 安装基础上新增：

- 首次向导后的股票搜索、自选、最近访问和工作区恢复；
- System / Light / Dark 主题、窄窗图标侧栏、键盘焦点和 200% 等效缩放验收；
- Formula Studio 桌面认证与 MACD / 自定义公式语义回归；
- sidecar 启动、健康、重启上限和确认退出的有界恢复；
- 用户显式保存、双重严格校验、不会自动上传的本地脱敏诊断包；
- 16–256 px 统一图标以及真实 packaged Tauri 的 exact-SHA Windows 自动截图证据。

SignPath Foundation 申请已提交并等待审核。本版本仍是 **unsigned prerelease**，没有 Authenticode 签名，Windows 可能显示“未知发布者”或 SmartScreen 提示。正式 `v1.1.0` 仍需可信签名和 Windows 10/11 普通用户真实 DPI / 全旅程验收。

本预发布不包含 macOS、Linux、Android 或 ARM64 资产，也不会迁移或删除 v1 本地数据。稳定使用请继续选择 [v1.0.0](https://github.com/CongBao/stock-desk/releases/tag/v1.0.0)。

## English

`v1.1.0-beta.2` is a Windows x64-only desktop-experience prerelease. It adds post-onboarding search and workspace recovery, System/Light/Dark themes, responsive icon navigation, authenticated Formula Studio semantics, bounded sidecar recovery, explicit local-only redacted diagnostic export, a multi-size Windows icon set, and exact-SHA evidence captured from the actual packaged Tauri candidate.

The SignPath Foundation application has been submitted and remains under review. This is still an **unsigned prerelease** without Authenticode signing, so Windows may show an Unknown Publisher or SmartScreen warning. Formal `v1.1.0` still requires trusted signing plus real Windows 10/11 standard-user DPI and full-journey acceptance.

This prerelease contains no macOS, Linux, Android, or ARM64 assets and does not migrate or delete v1 local data. Continue to use [v1.0.0](https://github.com/CongBao/stock-desk/releases/tag/v1.0.0) for the stable release.
