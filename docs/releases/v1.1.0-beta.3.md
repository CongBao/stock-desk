# Stock Desk v1.1.0-beta.3

[English](#english)

## 中文

`v1.1.0-beta.3` 是仅面向 Windows x64 的无签名桌面体验预发布版本。本版本在 beta.2 基础上新增和加强：

- 宽屏、窄屏、平板与低高度窗口的响应式交互，以及图标侧栏、键盘焦点和 200% 等效缩放覆盖；
- 退出确认、最长十秒的安全检查点等待，以及启动后的中断任务恢复或取消；恢复分析前会单独确认模型费用；
- 卸载时明确选择是否删除 **仅 v1.1** 的本地数据；取消删除或删除失败都会保留数据，旧版 v1 数据不在操作范围内；
- packaged Tauri 传输层回归覆盖 MACD / 自定义公式、单股 / 股票池、日线 / 周线 / 60 分钟的十二种组合；这不是完整 A 股边界规则的独立权威验证；
- 可信更新运行组件保持默认关闭，正式签名和稳定版发布控制面保持硬禁用。

本版本仍是 **unsigned prerelease**，没有 Authenticode 签名，Windows 可能显示“未知发布者”或 SmartScreen 提示。SignPath Foundation 申请仍在等待外部审核。它也不代表 Windows 10/11 普通用户完整安装旅程、SmartScreen / MOTW 或真实外部 VM 矩阵已经通过。

本预发布不包含 macOS、Linux、Android、ARM64 资产，也不会迁移或删除 v1 本地数据。稳定使用请继续选择 [v1.0.0](https://github.com/CongBao/stock-desk/releases/tag/v1.0.0)。

## English

`v1.1.0-beta.3` is an unsigned Windows x64-only desktop-experience prerelease. Compared with beta.2, it adds or strengthens:

- responsive interaction for wide, narrow, tablet-like, and short windows, including icon navigation, keyboard focus, and 200%-equivalent scaling coverage;
- exit confirmation, a bounded ten-second safe-checkpoint wait, and explicit resume or cancel after an interrupted launch; resuming analysis requires separate model-cost consent;
- an uninstall choice for removing **v1.1-only** local data; cancellation or deletion failure preserves the data, and legacy v1 data remains outside the operation;
- packaged-Tauri transport regression coverage across twelve MACD/custom-formula, single-symbol/pool, and daily/weekly/60-minute combinations; this is not an independent oracle for every A-share edge rule;
- a trusted-update runtime that remains default-off and formal signing/stable-release controls that remain hard-disabled.

This remains an **unsigned prerelease** without Authenticode signing, so Windows may show an Unknown Publisher or SmartScreen warning. The SignPath Foundation application remains under external review. This release is not proof that full Windows 10/11 standard-user installation journeys, SmartScreen/MOTW behavior, or a real external-VM matrix have passed.

This prerelease contains no macOS, Linux, Android, or ARM64 assets and does not migrate or delete v1 local data. Continue to use [v1.0.0](https://github.com/CongBao/stock-desk/releases/tag/v1.0.0) for the stable release.
