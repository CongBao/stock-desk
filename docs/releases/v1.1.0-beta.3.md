# Stock Desk v1.1.0-beta.3

[English](#english)

## 中文

`v1.1.0-beta.3` 是仅面向 Windows x64 的未签名桌面体验预发布版本。本版本在 beta.2 基础上新增和加强：

- 宽屏、窄屏、平板与低高度窗口的响应式交互，以及图标侧栏、键盘焦点和 200% 等效缩放覆盖；
- 退出确认、最长十秒的安全检查点等待，以及启动后的中断任务恢复或取消；恢复分析前会单独确认模型费用；
- 卸载时明确选择是否删除 **仅 v1.1** 的本地数据；取消删除或删除失败都会保留数据，旧版 v1 数据不在操作范围内；
- packaged Tauri 传输层回归覆盖 MACD / 自定义公式、单股 / 股票池、日线 / 周线 / 60 分钟的十二种组合；这不是完整 A 股边界规则的独立权威验证；
- 将 Tauri 2.11.4 使用的 NSIS 3.11 与 `nsis_tauri_utils` 0.5.3 绑定到经审计的官方下载摘要及完整提取树；
- 在受保护的私有快照内生成内容寻址的 NSIS 重打包工具包，拒绝绝对路径、未绑定文件、未知插件及可执行外部命令的编译期指令；
- 对同一 exact-SHA 候选执行两次独立重打包，要求字节完全一致并由严格回执绑定；
- main CI 通过后，Release 只复用不可变证明和候选，不重复运行源码测试或重新构建；
- 可信更新运行组件保持默认关闭，正式签名和稳定版发布控制面保持硬禁用。

本版本没有 Authenticode 签名，Windows 可能显示“未知发布者”或 SmartScreen 提示。SignPath Foundation 申请仍在等待外部审核。它也不代表 Windows 10/11 普通用户完整安装旅程、SmartScreen / MOTW 或真实外部 VM 矩阵已经通过。

本预发布不包含 macOS、Linux、Android、ARM64 资产，不连接券商、不执行交易，也不会迁移或删除 v1 本地数据。稳定使用请继续选择 [v1.0.0](https://github.com/CongBao/stock-desk/releases/tag/v1.0.0)。

## English

`v1.1.0-beta.3` is an unsigned Windows x64-only desktop-experience prerelease. Compared with beta.2, it adds or strengthens:

- responsive interaction for wide, narrow, tablet-like, and short windows, including icon navigation, keyboard focus, and 200%-equivalent scaling coverage;
- exit confirmation, a bounded ten-second safe-checkpoint wait, and explicit resume or cancel after an interrupted launch; resuming analysis requires separate model-cost consent;
- an uninstall choice for removing **v1.1-only** local data; cancellation or deletion failure preserves the data, and legacy v1 data remains outside the operation;
- packaged-Tauri transport regression coverage across twelve MACD/custom-formula, single-symbol/pool, and daily/weekly/60-minute combinations; this is not an independent oracle for every A-share edge rule;
- NSIS 3.11 and `nsis_tauri_utils` 0.5.3 used by Tauri 2.11.4 bound to audited official download digests and the complete extracted tree;
- a content-addressed NSIS repack kit created inside protected private snapshots, rejecting absolute paths, unbound files, unknown plugins, and compile-time instructions capable of external execution;
- two independent repacks of the exact-SHA candidate that must be byte-identical and are bound by strict receipts;
- Release reuse of the immutable proof and candidate after main CI succeeds, without rerunning source tests or rebuilding the installer;
- a trusted-update runtime that remains default-off and formal signing/stable-release controls that remain hard-disabled.

This release has no Authenticode signature, so Windows may show an Unknown Publisher or SmartScreen warning. The SignPath Foundation application remains under external review. This release is not proof that full Windows 10/11 standard-user installation journeys, SmartScreen/MOTW behavior, or a real external-VM matrix have passed.

This prerelease contains no macOS, Linux, Android, or ARM64 assets, does not connect to a broker or place trades, and does not migrate or delete v1 local data. Continue to use [v1.0.0](https://github.com/CongBao/stock-desk/releases/tag/v1.0.0) for the stable release.
