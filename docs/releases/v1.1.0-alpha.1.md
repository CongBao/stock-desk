# Stock Desk v1.1.0-alpha.1

[English](#english)

## 中文

`v1.1.0-alpha.1` 是交付基础预览，不包含 Tauri 桌面壳或新的安装体验。它保持 v1 行情、公式、回测和分析能力可运行，并为后续桌面版本建立：

- 保守、可审计的 PR 风险图；未知或高风险变更自动升级为全量门禁。
- `main` 四个 Python shard 的唯一测试归属、综合 branch coverage `>=85.00%` 和 exact-SHA JUnit 证据。
- 一次确定性 E2E snapshot/服务、一次 OCI 构建，以及内容寻址的 Web/Python/OCI 产物。
- 由本次 alpha release 验证并复用的不可变 main proof；alpha 不重新执行相同 unit/E2E，也不构建桌面安装包。该约束不表示现有稳定版流水线已经复用全部构建产物。

SignPath Foundation 申请状态为 `application-submitted / pending-review`。若发布 alpha 资产，它们必须标记为 **unsigned prerelease**，不代表已有 Authenticode 签名或通过 SmartScreen 正式门禁。

详细契约见 [CI 与不可变交付证据](../ci.md)。稳定版安装请继续使用 [v1.0.0](https://github.com/CongBao/stock-desk/releases/tag/v1.0.0)。

## English

`v1.1.0-alpha.1` is a delivery-foundation preview. It does not include the Tauri desktop shell or the new installation experience. It keeps the v1 market, formula, backtest, and analysis capabilities runnable while establishing:

- A conservative, auditable PR risk graph that escalates unknown or high-risk changes to full gates.
- Unique test ownership across four Python `main` shards, combined branch coverage `>=85.00%`, and exact-SHA JUnit evidence.
- One deterministic E2E snapshot/service, one OCI build, and content-addressed Web/Python/OCI artifacts.
- An immutable main proof verified and reused by this alpha release. The alpha neither reruns the same unit/E2E tests nor builds desktop installers; this does not claim that the existing stable-release path already reuses every build output.

The SignPath Foundation status is `application-submitted / pending-review`. Any alpha assets must be labelled **unsigned prerelease** and do not claim Authenticode signing or formal SmartScreen acceptance.

See [CI and immutable delivery evidence](../ci.md) for the contracts. Continue using the [v1.0.0 release](https://github.com/CongBao/stock-desk/releases/tag/v1.0.0) for stable installation.
