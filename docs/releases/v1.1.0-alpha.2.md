# Stock Desk v1.1.0-alpha.2

[English](#english)

## 中文

`v1.1.0-alpha.2` 是仅面向 Windows x64 的桌面壳预发布版本，包含由 Tauri 承载的
独立桌面窗口、受控 Python sidecar、单实例与启动恢复，以及无需管理员权限的当前用户
NSIS 安装器。安装器直接复用同一提交在 `main` 双独立 builder 比对后提升的候选，
Release 只验证不可变 proof、attestation、manifest 与校验和，不重新运行源码测试，
也不重新构建安装器。

SignPath Foundation 申请仍为 `application-submitted / pending-review`。本版本是
**unsigned prerelease**，没有 Authenticode 签名；Windows 可能显示“未知发布者”或
SmartScreen 提示。它不是稳定版或最新版。请仅在接受该边界的测试环境使用；
正式 `v1.1.0` 仍需可信签名和 Windows 10/11 普通用户验收。

本预发布不包含 macOS、Linux、Android 或 ARM64 资产。稳定使用请继续选择
[v1.0.0](https://github.com/CongBao/stock-desk/releases/tag/v1.0.0)。

## English

`v1.1.0-alpha.2` is a Windows x64-only desktop-shell prerelease. It contains a
standalone desktop window hosted by Tauri, controlled Python sidecar, single-instance and startup
recovery behavior, and a per-user NSIS installer that does not require an
administrator. The installer is the exact candidate promoted after two
independent builders were compared on `main`; Release verifies the immutable
proof, attestations, manifests, and checksums without rerunning source tests or
rebuilding the installer.

The SignPath Foundation application remains `application-submitted /
pending-review`. This is an **unsigned prerelease** without Authenticode signing,
so Windows may show an Unknown Publisher or SmartScreen warning. It is neither a
stable nor latest release. Use it only in a test environment that accepts this
boundary; formal `v1.1.0` still requires trusted signing and Windows 10/11
standard-user acceptance.

This prerelease contains no macOS, Linux, Android, or ARM64 assets. Continue to
use [v1.0.0](https://github.com/CongBao/stock-desk/releases/tag/v1.0.0) for the
stable release.
