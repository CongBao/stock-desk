# Support

## Questions

Search the [documentation](README.md) and existing issues first. Community
support is best-effort; the project does not provide guaranteed response times,
individualized trading guidance, market-data credentials, or investment advice.

## Bug reports

For a reproducible bug or feature proposal, use the
[GitHub issue chooser](https://github.com/CongBao/stock-desk/issues/new/choose)
and select the matching template. Include the Stock Desk version, operating
system, Python/Node versions, run mode, minimal reproduction, task ID, and
sanitized logs.

Before sharing diagnostics, remove secrets, account identifiers, personal data,
licensed data samples, database contents, backups, local paths, and model prompts
containing private material. The [troubleshooting guide](docs/troubleshooting.md)
lists the useful non-sensitive context for common failures.

## Security

Security vulnerabilities belong in
[GitHub private vulnerability reporting](https://github.com/CongBao/stock-desk/security/advisories/new),
never a public support issue. Follow [SECURITY.md](SECURITY.md) and do not include
an active credential in the report.
