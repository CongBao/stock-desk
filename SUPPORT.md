# Support

For reproducible bugs and feature proposals, use the [GitHub issue chooser](https://github.com/CongBao/stock-desk/issues/new/choose) and select the matching template. Search existing issues first and include the Stock Desk version, operating system, Python/Node versions, run mode, minimal reproduction, and sanitized logs.

Community support is best-effort. This project does not provide guaranteed response times, individualized trading guidance, market-data credentials, or investment advice.

Security vulnerabilities belong in [GitHub private vulnerability reporting](https://github.com/CongBao/stock-desk/security/advisories/new), never a public support issue. Remove secrets, account identifiers, personal data, and licensed data samples before sharing any diagnostic material.
